## Technical Implementation

### State Machine
```typescript
type OnboardingStep =
  | 'splash'
  | 'opening_animation'
  | 'memories_emerging'
  | 'manifesto'
  | 'capsule_selection'
  | 'capsule_explanation'
  | 'registration'
  | 'legal'
  | 'payment'
  | 'capsule_active';

interface OnboardingState {
  currentStep: OnboardingStep;
  selectedCapsuleType: CapsuleType | null;
  isAuthenticated: boolean;
  legalAccepted: boolean;
  subscriptionTier: SubscriptionTier;
}
```

### Animation Specifications

#### Splash â†’ Opening
- Capsule: 3D model with metallic shader
- Tap triggers lid lift animation (ease-out)
- Light particles emerge from opening
- Duration: 2s opening, 0.5s settle

#### Memories Emerging
- Polaroid-style images (3-5)
- Float upward with gentle rotation
- Staggered timing (0.3s apart)
- Fade in as they rise
- Duration: 3s total

#### Manifesto
- Text fades in sequentially
- Elegant serif font (Cormorant Garamond)
- Subtle background movement
- Button appears after text complete

### Registration Integration

#### Supabase Auth
```typescript
// Apple Sign In
const { data, error } = await supabase.auth.signInWithOAuth({
  provider: 'apple',
  options: {
    redirectTo: `${window.location.origin}/auth/callback`
  }
});

// Google Sign In
const { data, error } = await supabase.auth.signInWithOAuth({
  provider: 'google',
  options: {
    redirectTo: `${window.location.origin}/auth/callback`
  }
});

// Email/Password
const { data, error } = await supabase.auth.signUp({
  email,
  password,
  options: {
    data: {
      selected_capsule_type: selectedCapsuleType
    }
  }
});
```

### Skip Conditions

| From Step | Skip To | Condition |
|-----------|---------|-----------|
| splash | capsule_selection | Returning user, skip animation |
| registration | legal | Already authenticated |
| payment | capsule_active | Free tier capsule selected |
| legal | capsule_active | Already accepted (returning user) |

### Persistence

```typescript
// Store onboarding progress
localStorage.setItem('nuclea_onboarding', JSON.stringify({
  completed: false,
  lastStep: 'manifesto',
  selectedCapsuleType: 'legacy',
  timestamp: Date.now()
}));

// Mark as completed
const markOnboardingComplete = () => {
  localStorage.setItem('nuclea_onboarding', JSON.stringify({
    completed: true,
    completedAt: Date.now()
  }));
};
```

## Metrics to Track

| Event | Description |
|-------|-------------|
| `onboarding_started` | Tap on splash screen |
| `onboarding_manifesto_viewed` | Reached manifesto step |
| `capsule_type_selected` | User selected a capsule type |
| `registration_started` | Entered registration step |
| `registration_method` | Which auth method used |
| `registration_completed` | Successfully created account |
| `legal_accepted` | Accepted terms |
| `payment_started` | Entered payment step |
| `payment_completed` | Successful payment |
| `onboarding_completed` | Reached capsule active state |
| `onboarding_abandoned` | Left before completion (with step) |
