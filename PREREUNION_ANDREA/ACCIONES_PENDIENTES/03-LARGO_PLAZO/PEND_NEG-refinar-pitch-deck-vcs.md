# Refinar Pitch Deck para VCs

## Descripción

Crear/refinar pitch deck específico para presentaciones a VCs y business angels. Versión más detallada que el one-pager.

## Criterios de Éxito

- [ ] Estructurar 10-12 slides clásicos
- [ ] Incluir datos de mercado verificados
- [ ] Añadir roadmap de producto
- [ ] Incluir proyecciones financieras básicas
- [ ] Diseñar con branding NUCLEA
- [ ] Preparar versión PDF y presentación

## Estructura Propuesta

| Slide | Contenido |
|-------|-----------|
| 1 | Cover (tagline + logo) |
| 2 | El Problema (duelo digital, legado perdido) |
| 3 | La Solución (5 cápsulas + avatar AI) |
| 4 | Diferenciador (consentimiento firmado) |
| 5 | Mercado (TAM/SAM/SOM con fuentes) |
| 6 | Competencia (mapa + tabla comparativa) |
| 7 | Modelo de Negocio (pricing, unit economics) |
| 8 | Producto (demo/wireframes) |
| 9 | Traction (usuarios, waitlist, métricas) |
| 10 | Roadmap (6-12 meses) |
| 11 | Equipo (Andrea + Imanol) |
| 12 | Ask (qué buscamos, uso de fondos) |
