# WORKFLOYS

Flujos de trabajo automatizados para NUCLEA.

## Flujo de Desarrollo Estandar

```
1. ANALISIS
   └─ /autothink [tarea]     → Clarificar requerimiento

2. PLANIFICACION
   └─ /writing-plans         → Crear plan de implementacion

3. IMPLEMENTACION
   ├─ /delegate cy "tarea"   → Claude para complejo
   ├─ /delegate both "tarea" → Paralelo para speed
   └─ /rapid-prototype       → MVP rapido

4. REVIEW
   ├─ /requesting-code-review → Verificar calidad
   └─ /webapp-testing         → Testing automatico

5. DOCUMENTACION
   ├─ /htmlreport "topic"    → Reporte HTML
   └─ /ascii "topic"         → Reporte ASCII
```

## Flujo de Investigación

```
/jarvis "topic"         → 5 busquedas encadenadas
   ↓
/jarvisyt "topic"       → + 2 transcripts YouTube
   ↓
/htmlreport "resultado" → Generar reporte
```

## Flujo de Debugging

```
1. /systematic-debugging → Analizar bug
2. /think-harder         → Analisis profundo
3. /qaterminal "topic"   → Evaluar conocimiento
```

## Flujo de Documentación

```
1. /doc-coauthoring      → Guiar creacion
2. /sidebar              → Generar con sidebar
3. /eureka               → Capturar insights
```
