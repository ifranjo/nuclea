# PROMPTS_MULTI

Prompts listos para usar con `ky`, `cy` y `codex` en NUCLEA.

## Uso rapido

1. Ejecuta implementacion con `ky`.
2. Ejecuta review/riesgo con `cy`.
3. Ejecuta cierre tecnico con `codex`.

## Comandos sugeridos

```bash
/delegate ky "<pegar PROMPT_KY_P0_EXECUTION.md>"
/delegate cy "<pegar PROMPT_CY_GATE_REVIEW.md>"
```

Para `codex`, pegar el contenido de `PROMPT_CODEX_DIRECT_EXECUTION.md` directamente en la sesion.
