Executive Summary

NUCLEA es una plataforma de capsulas digitales para preservacion emocional con AI avatar opcional. Estamos construyendo la primera solucion en espanol que permite a las personas preservar sus memorias, historias y legado digital de forma estructurada y emocional.

El Problema: El legado digital esta fragmentado. Las fotos estan en moviles, videos en la nube, cartas en cajones. No existe una solucion hispanohablante integrada para preservacion emocional digital. El 68% de las personas no tienen ningun tipo de plan de sucesion, y el 80% de quienes si lo tienen no incluyen sus activos digitales.
La Solucion: 5 tipos de capsulas digitales (EverLife, Life Chapter, Social, Pet, Origin) que permiten preservar memorias con formato estructurado. Diferenciador unico: avatar AI con consentimiento explicito firmado en vida.

interface User {
  id: string

  displayName: string
  photoURL?: string

  createdAt: Timestamp
  capsuleCount: number
  storageUsed: number        // bytes

    signed: boolean

// capsules/{capsuleId}
interface Capsule {
  id: string
  userId: string             // ref: users/{userId}
  type: 'everlife' | 'life-chapter' | 'social' | 'pet' | 'origin'
  title: string
  description: string
  coverImage?: string        // URL Storage
  createdAt: Timestamp
  updatedAt: Timestamp
  isPublic: boolean
  sharedWith: string[]       // array de userIds
  scheduledRelease?: Timestamp
  status: 'draft' | 'active' | 'sealed' | 'released'
  tags: string[]
}
